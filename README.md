# Zabbix + Grafana Home Lab

Este projeto documenta a implementação de um ambiente de monitoramento de rede e serviços utilizando **Zabbix** e **Grafana**, executados em uma máquina virtual com **Ubuntu Server**.

## Objetivo
Criar um laboratório funcional de monitoramento para:
- Dispositivos de rede
- Hosts e serviços
- Métricas de desempenho
- Visualização em dashboards

## Arquitetura
- Máquina Virtual (Hypervisor local)
- Ubuntu Server LTS
- Zabbix Server
- Grafana
- Monitoramento via SNMP e Agent

## Tecnologias Utilizadas
- Ubuntu Server
- Zabbix
- Grafana
- SNMP
- Virtualização
